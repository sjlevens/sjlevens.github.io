Quarkus Panache/Hibernate/CRUD

I lost my mind looking for tutorials

So I am making my own

List of technologies used

  • Java 11 (openjdk version "11.0.11" 2021-04-20)
  • Quarkus 2.0
  • Hibernate
  • Panache
  • Resteasy
  • Jackson
  • PostgresQL

What are we going to make

An endpoint that is connected to a PostgresQL DB

This endpoint will allow CRUD operations of a few types of entities

These entities will have relationships that are Many to One, One to Many, Many to Many and One to One

Why these technologies

They are new, interesting and actually really cool once you get the hang of how they work.

Panache particularly is going to make this all really clean. Most of the tutorials I found were using PanacheBaseEntity but we will be using the full Panache power!
